

## Fronted para el sistema de notas. 

#Instalación 
  Pasos a seguir para la instalación del proyecto

  * Instalar las dependencias 
  ```console
    npm install
  ```

  * Ejectuar el sistema
  ```console
    npm run dev
  ```
